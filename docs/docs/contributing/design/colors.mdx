# Colors

The palette includes the primary brand colors as well as two complimentary tones in a range of tints and shades for use on the website, packaging, and additional brand materials. When using with text, all color pairings should have adequate contrast in accordance with [WCAG 2.0 guidelines on contrast ratio](https://contrast-ratio.com).

## Palette

export const Swatch = function ({ color, name }) {
  const rgb = color.match(/([1-9])\w+/g)
  const hex = rgb.reduce((str, val) => {
    str += parseInt(val).toString(16)
    return str
  }, '#')
  return (
    <div className={`stmx-color-swatch stmx-brand-color-${name}`}>
      <div className="line-block">
        <div className="line">
          <strong>{name.split('-')[1]}</strong>
        </div>
        <div className="line">
          <strong>R</strong> {rgb[0]}
        </div>
        <div className="line">
          <strong>G</strong> {rgb[1]}
        </div>
        <div className="line">
          <strong>B</strong> {rgb[2]}
        </div>
      </div>
      <p>{hex}</p>
    </div>
  )
}
